### Preguntas 
1. Si tuviera que implementar una estructura tipo Cola (Queue), ¿qué tipo de lista (simple, circular, doblemente enlazada o circular doblemente enlazada) utilizaría y por qué? ¿Y para una estructura tipo Pila (Stack)?

R/ Para la Cola Queue usaría una lista circular simple, ya que el último nodo puede señalar al primero y así se puede mantener un flujo continuo. En cambio para la estructura tipo Pila usaría una lista enlazada, ya que solo se necesita trabajar con lado que seria el tope de la pila. 

---

2. ¿Cuál cree que podría ser una ventaja y una desventaja de utilizar una estructura de tipo lista creada por usted mismo, en vez de las opciones proveídas directamente por los módulos estándar de Java (como por ejemplo List)?


R/ Una ventaja es que al hacer la lista uno mismo, se entiende mejor cómo funcionan los nodos, los punteros next y prev, y cómo se enlazan los datos.

Una desventaja es que puede ser más fácil cometer errores y lleva más tiempo hacerla, mientras que las listas que ya trae Java (ArrayList, LinkedList, etc.) ya están listas para usar y son más seguras y rápidas.
